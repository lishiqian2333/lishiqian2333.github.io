web3使用

链接到区块链网络

const Web3 = require('web3') //加载web3库
const rpcURL = "https://goerli.infura.io/v3/705170723e474c33950e5044643883a9" //创建web3连接
const web3 = new Web3(rpcURL)

查地址余额

web3.eth.getBalance(address, (err, wei) => {
    // 余额单位从wei转换为ether
    balance = web3.utils.fromWei(wei, 'ether')
    console.log("balance: " + balance)
})

智能合约对象
对象可以使用web3.eth.Contract()函数获得，此函数需要2个参数: 智能合约ABI、智能合约地址

const abi = [合约abi]
const address = 合约地址
const contract = new web3.eth.Contract(abi, address)

智能合约函数的调用
智能合约对象的methods属性下，包含了对应智能合约的所有函数。要调用智能合约中的某个函数，例如myFunction()，可以使用contract.methods.myFunction()的方式调用
